# Vue 3 + TypeScript + Vite

### Vue 3 项目核心工具与库
1. element-plus 组件库
2. UnoCSS 原子化 CSS 引擎
3. Axios 网络请求库
4. vue-router 路由
5. layout 布局
6. unplugin-vue-components 自动导入组件
7. unplugin-auto-import 自动导入API
8. vite-plugin-vue-layouts 布局插件
9. eslint 代码规范


安装  
`npm install`  

启动
`npm run dev`  

打包
`npm run build`
